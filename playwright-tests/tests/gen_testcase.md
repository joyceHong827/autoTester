# AI 產生測試報告範例

以下是一個由 AI 生成的測試報告範例，展示了如何描述測試過程、結果和結論。

## 步驟
1. 建立一個空資料夾專門放測試報告檔案
2. 建立一份測試報告的範本以供 AI 參考 (可參考 )
3. 在該資料夾中建立一個新的 Markdown 檔案 (例如: `gen_testcase.md`)
4. 開啟  agent 模式，選擇合適的模型
5. 輸入指令讓 AI 生成測試報告內容，如下所示:
   ```
   請閱讀 redmine [單號]，再幫我寫測試報告，內容包含測試目的、測試步驟、測試資料、預期結果、實際結果、驗證項
   目、測試結論等，並以 Markdown 格式呈現, 驗證項目用表格呈現，請勿有emoji的表情符號。
   ```
5.1 簡化版:
   ```
   請閱讀 redmine [單號]，再幫我寫測試報告
   ```
5. 閱讀並確認生成的測試報告內容是否符合需求
6. 若有需補充或修改的部分，可再次輸入指令讓 AI 進行調整
