## 測試目的

1. 登入 B2E 後台
2. 進入「電視商品管理 > 商品開發&審核」(https://b2e.etmall.com.tw/Web/TVProduct)

2. 進入「電視商品管理 > 商品開發&審核」
3. 點擊「商開審核」按鈕，載入商品列表
4. 點選電視商品，開啟商品明細頁
5. 嘗試點擊「ESV價」欄位並輸入數值

**預期結果**:
- ESV 價欄位可正常輸入數值（商開審核作業時開放修改）
- 輸入 9900 後欄位值顯示為 9900

### 情境 5：ESV 欄位不寫至富購系統

**測試步驟**:
1. 確認電視商品的 ESV 價已儲存於資料庫（DraftCommon.ESVPrice 有值）
2. 執行富購同步作業
3. 確認富購系統中對應商品是否包含 ESVPrice 欄位

**預期結果**:
- 富購系統中不包含 ESVPrice 欄位或其值未被同步寫入
- 其他欄位同步不受影響
